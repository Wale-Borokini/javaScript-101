Languages like PHP, RUBY run on a server somewhere else. but Javascript runs on our computer. It is object Based. It is an object oriented programmer. It is prototype based instead of class based. JavaScript is NOT java. They have a similar name but are completely different. Java is much more complex. Java is a high level language that is in the same category as C and C++.
